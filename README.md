# Studora

Studora is a student productivity and management app built using Kotlin and Jetpack Compose. It helps students manage timetables, tasks, assignments, and attendance through a modern, user-friendly interface.

## Features

* Timetable Management
* Task & Assignment Tracking
* Attendance Monitoring
* Student Dashboard
* Priority-based Task Management
* Modern Dark UI
* Bottom Navigation
* Student Profile Management

## Tech Stack

* Kotlin
* Jetpack Compose
* Android Studio
* Material Design
* Room Database

## Project Screens

### Dashboard

Provides a quick overview of attendance, pending tasks, upcoming classes, and deadlines.

### Timetable

Manage daily academic schedules with an organized timetable interface.

### Tasks

Track assignments and reminders using priority-based task management.

### Attendance

Monitor overall and subject-wise attendance statistics.

### Profile

Manage personal information and application settings.

## Project Goal

The goal of Studora is to simplify student life by combining timetable management, attendance tracking, task organization, and academic planning into a single mobile application.
