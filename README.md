## Features

- Timetable Management
- Task & Assignment Tracking
- Attendance Monitoring
- Student Dashboard
- Priority-based Task Management
- Modern Dark UI
- Bottom Navigation

## Tech Stack

- Kotlin
- Jetpack Compose
- Android Studio
- Material Design
